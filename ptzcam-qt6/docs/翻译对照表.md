## 如何新增语言

1. 在 `app/utils/i18n.py` 的 `_TRANSLATIONS` 字典中，为每个词条添加新语言的 key
2. 示例：
```python
"就绪": {"zh": "就绪", "en": "Ready", "ja": "準備完了"},
```
3. 更新 `set_language(lang)` 接受新的语言代码
4. 在语言 ComboBox 中添加新选项
